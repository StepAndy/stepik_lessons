1. Все товары - http://selenium1py.pythonanywhere.com/ru/catalogue/
	1.1. Поиск товара
		1.1.1. Поиск по подстроке
        Шаги:
        1. Кликнуть на окно поиска

        2. В окне поиска ввести название товара из каталога (прим. handbook).
        ОР: Placeholder исчизает при вводе.

        3. Кликнуть кнопку "Найти".
        ОР: Найдены все товары с подстрокой, указано верное число результатов, присутствует функция сортировки.

	1.2. Добавление товара в корзину
		1.2.1. Добавить товара из каталога (не из карточки товара)
		Шаги:
		1. Кликнуть "Добавить в корзину"
		ОР: На странице появляются алерты о добавлении выбранного товара в корзину
		и удовлетворении условий для Shipping offer

2. Корзина - http://selenium1py.pythonanywhere.com/ru/basket/
	2.1. Скидка по промокоду
		2.1.1. Получение 50% скидки
		Предусловия:
		В корзину добавлен любой товар

		Шаги:
		1. Кликнуть кнопку активации промокода.
		ОР: Появляется диалоговое окно с полем ввода промокода

		2. Ввести промокод с 50% скидкой

		3. Кликнуть "Применить"
		ОР: Цена на товары в корзине указана с учетом скидки

	2.2. Оформить покупку
		2.2.1. Оформить покупку для нового пользователя
        Предусловия:
        В корзину добавлен любой товар
        Осуществлен переход к оформлению покупки

        Шаги:
        1. Указать адрес электронной почты и кликнуть "Продолжить".
        ОР: Поле визуально обозначено как обязательное (*), проверяется формат электронной почты, переход на следующий шаг

        2. Заполнить все обязательные поля формы "Адрес доставки" валидными значениями и кликнуть "Продолжить"
        ОР: переход на следующий шаг

        3. На странице "Параметры платежа" кликнуть продолжить
        ОР: переход на следующий шаг

        4. Посмотреть данные заявки
        ОР: в заявке указаны введенные на предыдущих шагах данные

        5. Кликнуть разместить заказ
        ОР: На странице предпросмотра появляется информация о подтверждении заказа,
        отображаются кликабельные кнопки для проверки статуса заказа, печати страницы, возврата к покупкам
